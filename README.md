# Dr. Manpreet Kaur — Personal Academic Website

Personal website for Dr. Manpreet Kaur, Postdoctoral Researcher at IISc Bengaluru.

## Stack
Static HTML/CSS/JS — no build step required. Hosted via GitHub Pages.

## Setup (GitHub Pages)

1. Push all files to the `main` (or `gh-pages`) branch of `https://github.com/mkaur-quant`
2. Go to **Settings → Pages**, set source to **Deploy from branch → main → / (root)**
3. The site will be live at `https://mkaur-quant.github.io`

## Add your CV PDF
Place your CV as `CV_Manpreet_Kaur.pdf` in the root folder — the Download CV button will pick it up automatically.
